# Object Detection and Tracking System

## Description

This project uses YOLOv8 and Deep SORT to detect and track objects in a video.

## Features

* Real-time object detection
* Multi-object tracking
* Unique tracking IDs
* Output video generation

## Technologies

* Python
* OpenCV
* YOLOv8
* Deep SORT

## Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run:
   python main.py

## Output

The processed video is saved in the outputs folder.
